# Web Development Studio Landing Page

A modern, minimalistic landing page built with React and Tailwind CSS, featuring a dark Miami/sci-fi aesthetic.

## Features

- **Hero Section** with headline, CTA buttons, and UI mockup
- **What We Do** section with 5 service cards
- **Portfolio Previews** with 4 sample projects
- **Why Choose Us** section with 3 benefit cards
- **Pricing Section** with 3 pricing tiers
- **Contact Section** with form and Calendly integration card
- **Footer** with copyright and tech stack info

## Tech Stack

- React 18
- Tailwind CSS 3
- Vite

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Design Notes

- Dark theme with gradient backgrounds
- Miami-inspired color palette (cyan, pink, purple)
- Rounded cards with subtle borders and shadows
- Responsive design for all screen sizes
- Smooth hover transitions and effects
